¿que es HTMLy para que sirve?

HTML es un lenguaje basico de marcado de la www(worl wide web). es un estandar que se toma de referencia para la definicion de una pagina web, con capacidad para texto, video, imagenes,juegos,entre otros.es considerado el lenguaje mas importate y fundmental en la expancion de la www

¿que es css3 y como funciona?

sus siglas en ingles son (Cascading Sheet System) que traducido en español significa Hojas de estilo en cascada. es un lenguaje de diseño grafico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. es frecuentemente usado para establecer el diseño visual de documentos web escritas en leguaje html. CSS está diseñado principalmente para marcar la separación del contenido del documento y la forma de presentación de este, características tales como las capas o layouts, los colores y las fuentes.

¿que es un estilo externo?

un estilo interno es una forma de insertar diseños y caracteristicas a una pagina a partir desde un enlace CSS, facilitando la practicidad del diseño.

¿que es un estilo interno?

un estilo inteno es una forma de insertar o crear diseñoa a una pagina dando caracteristicas definiendo cada estilo directamente al objeto que se le desea insertar.
